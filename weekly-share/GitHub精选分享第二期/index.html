edex-ui，一个到随处运行的科幻桌面

eDEX-UI 是一个全屏桌面应用程序，类似于科幻电脑界面，
其灵感来自于DEX-UI和TRON Legacy电影特效。
它在真实终端中运行您选择的shell，并显示有关系统的实时信息。

Howler.js - JavaScript 开发跨平台音频相关的开发

Howler.js提供了一个现代音频库，支持Web Audio API和HTML5 Audio的回退机制。
该项目致力于简化与使用 JavaScript 开发跨平台音频相关的开发工作。
几家大型技术和媒体公司正在使用 Howler.js，包括谷歌、迪士尼、Mozilla 和乐高。
